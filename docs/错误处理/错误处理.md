# 错误处理


## 固件加载报错

```text

[    6.729846] bluetooth hci0: Direct firmware load for rtl_bt/rtl8723d_fw.bin failed with error -2

```

解决方法：

```shell
sudo apt update
sudo apt install firmware-realtek

```

## rk3288-crypto 报错

```text
[    7.374506] rk3288-crypto: probe of ff8b8000.crypto failed with error -16

```

解决措施：

临时禁用

```text
&crypto1 {
	status = "disabled";
};

```
